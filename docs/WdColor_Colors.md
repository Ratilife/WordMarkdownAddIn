## Использование в коде

Пример использования цветов в C#:

```csharp
using Word = Microsoft.Office.Interop.Word;

// Установка цвета текста
range.Font.Color = Word.WdColor.wdColorBlue;

// Использование в словаре для подсветки синтаксиса
private static readonly Dictionary<TokenType, Word.WdColor> TokenColors =
    new Dictionary<TokenType, Word.WdColor>
{
    [TokenType.Keyword] = Word.WdColor.wdColorBlue,
    [TokenType.String] = Word.WdColor.wdColorGreen,
    [TokenType.Comment] = Word.WdColor.wdColorGray25,
    [TokenType.Number] = Word.WdColor.wdColorOrange,
    [TokenType.Operator] = Word.WdColor.wdColorDarkRed,
    // ...
};
```

## Примечания

- **wdColorAutomatic** — специальное значение, которое использует цвет по умолчанию (обычно черный)
- Все цвета представлены в формате RGB (24-битное значение)
- Десятичное значение может быть отрицательным (как в случае с `wdColorAutomatic`)
- Некоторые цвета могут иметь одинаковые RGB значения, но разные имена (например, `wdColorTan` и `wdColorLightOrange`)
